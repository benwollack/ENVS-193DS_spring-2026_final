# ENVS-193DS_spring-2026_final

## General Information

This repo contains information for the final assessment for ENVS 193DS, including a revision of a statistical report's data interpretation (Problem 1), an analysis of the relationship between acacia tree height, ant species in the tree and the probability of a bird nest being in the tree (Problem 2) and an analysis of my personal data (Problem 3).

## Data & File Overview

The `code` folder contains the code for this assessment.

The `data` folder contains the data used, including the nest data and my personal data.
